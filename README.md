# dapp-security-ton

Прототип инструментального комплекса для оценки защищенности DApps в сети TON.

## Stack
- TypeScript
- Tact
- TON SDK

## Current modules
- static analyzer
- deployment scripts
- contract interaction scripts

## Project structure
- `src/` — исходный код анализатора
- `scripts/` — служебные скрипты deploy/hack
- `contracts/` — Tact-контракты
- `docs/` — документы по дипломной работе
- `configs/` — конфигурации
- `data/` — примеры и датасеты
